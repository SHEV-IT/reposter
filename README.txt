# reposter
This thing was cheated for automatical repost from vk social network to telegram.
Before launch add to file "constants.py" your vk id or vkgroup id with "-", telegram bot token and telegram chat id.
For launch copy all to some filder and write in console "python3 repeater.py".
Now for every 5 minute you check vk.com for new posts and in case when there will be some new it will be posted in telegram/
